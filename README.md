# 📝 Notes App

A fast, lightweight notes application built with vanilla JavaScript — shipped from idea to working app using AI-assisted development.

> A clean example of vibe coding in action: scoped the problem, used AI tools to accelerate build, iterated on UX until it felt right.

---

## ✨ Features

- ➕ Create, edit, and delete notes instantly
- 💾 Persistent storage — notes survive page refresh
- 🎨 Clean, minimal UI — built with HTML/CSS/JS, no frameworks
- ⚡ Zero dependencies — runs in any browser

---

## 🛠️ Tech Stack

`HTML` · `CSS` · `JavaScript` · `localStorage`

---

## ⚡ Quick Start

```bash
# Clone the repo
git clone https://github.com/onikeyek/notes-app.git

# Open in browser
open notes-app/index.html
```

No build step. No installs. Just open and use.

---

## 💡 Why This Project

Built to practice shipping a complete, usable product end-to-end — fast. The focus was on clean UX and persistent state management without reaching for a framework. Demonstrates the ability to scope a problem, build it lean, and deliver something that actually works.
